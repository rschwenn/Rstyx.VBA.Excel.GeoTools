Das Add-In
   

Verzeichnisse
Das Add-In “GeoTools.xlam” kann wie jedes andere Add-In installiert werden.
Ø             Add-In:  GeoTools.xlam = AddIn-Verzeichnis
Ø             Konfiguration:  GeoTools_cfg.xlsx = Unterverzeichnis "GeoToolsRes"
Ø             Dokumentation:  GeoTools.chm = Unterverzeichnis "GeoToolsRes"
Ø             Protokoll-Add-In 32 bit Actions.NET.x32.xll = Unterverzeichnis "GeoToolsRes"
Ø             Protokoll-Add-In 64 bit Actions.NET.x64.xll   = Unterverzeichnis "GeoToolsRes"

Das Protokoll-Add-In ist optional und wird beim Start der GeoTools automatisch geladen. In Abhängigkeit der Office-Bitbreite kommt die eine oder andere Variante zum Einsatz.

Weitere Abhängigkeiten
Einige Routinen stützen sich auf den Windows Script Host. Deshalb ist ein korrekt installiertes Windows Scripting unabdingbar für die Verwendung der GeoTools.
